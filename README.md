# Node Practice (nodejs-hw)

a collection of Node.js homework and practice projects. Includes backend and fullstack exercises covering REST API, authentication, database operations, and integration with frontend frameworks like React and Next.js.

## Branches

- [HW-1](https://github.com/Alina29051981/nodejs-hw/tree/01-express) – basic Node.js exercises (files, modules, scripts)  

- [HW-2](https://github.com/Alina29051981/nodejs-hw/tree/02-mongodb) – Express server with REST API  
  
- [HW-3](https://github.com/Alina29051981/nodejs-hw/tree/03-validation) – CRUD with MongoDB and Mongoose  
 
- [HW-4](https://github.com/Alina29051981/nodejs-hw/tree/04-auth) – JWT authentication and protected routes  
 
- [HW-5](https://github.com/Alina29051981/nodejs-hw/tree/05-mail-and-img) – Fullstack project with Node.js, Express, MongoDB, React/Next.js
